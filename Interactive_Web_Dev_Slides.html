🚀 What is Web Development?

Web development is building and maintaining websites and web applications that users interact with in their browsers.

🎨 Frontend

What users see and interact with

HTML, CSS, JavaScript

🔧 Backend

Server logic and data management

Databases, APIs, Servers

🔄 Full Stack

Both frontend and backend

Complete applications

How Browsers Work:

  • ✓ User enters a URL
  • ✓ Browser requests files from server
  • ✓ Server sends HTML, CSS, JavaScript
  • ✓ Browser renders the page
  • ✓ JavaScript makes it interactive

📝 HTML Fundamentals

HTML (HyperText Markup Language) provides the structure of web pages using tags.

Basic Structure

<!DOCTYPE html> <html lang="en"> <head> <title>My Website</title> </head> <body> <h1>Welcome!</h1> <p>This is content</p> </body> </html>

Common HTML Tags

Tag Purpose Example
<h1> - <h6> Headings <h1>Title</h1>
<p> Paragraph <p>Text</p>
<a> Link <a href="url">Link</a>
<button> Button <button>Click</button>

🎨 CSS Fundamentals

CSS (Cascading Style Sheets) provides styling and layout for HTML elements.

CSS Syntax

/* Selector */ h1 { /* Property: Value; */ color: blue; font-size: 48px; text-align: center; }

CSS Selectors

Selector Purpose Example
Element All elements of type p { }
Class Elements with class .highlight { }
ID Element with ID #header { }
Pseudo-class Special state a:hover { }

🎨 CSS Box Model & Flexbox

Box Model

Every element has: Margin → Border → Padding → Content

.box { width: 200px; padding: 20px; /* Inside */ border: 2px solid #333; margin: 10px; /* Outside */ background-color: lightblue; }

Flexbox (Modern Layout)

.container { display: flex; justify-content: center; /* Horizontal */ align-items: center; /* Vertical */ gap: 20px; /* Space between */ }
  • Flexbox makes responsive layouts easy
  • Great for centering and distributing items
  • Works on modern browsers

    • ⚙️ JavaScript Fundamentals

    JavaScript adds interactivity and logic to web pages.

    Variables & Data Types

    let name = "Alice"; // String const age = 25; // Number let isStudent = true; // Boolean let scores = [95, 87, 92]; // Array let user = { // Object name: "Bob", age: 30 };

    Functions

    function greet(name) { return "Hello, " + name; } // Arrow Function (Modern) const add = (a, b) => a + b;

    ⚙️ Conditionals & Loops

    If Statements

    if (age >= 18) { console.log("Adult"); } else if (age >= 13) { console.log("Teenager"); } else { console.log("Child"); }

    Loops

    // For Loop for (let i = 0; i < 5; i++) { console.log(i); } // Array forEach items.forEach(item => { console.log(item); });

    🖱️ DOM & Event Handling

    The DOM (Document Object Model) lets JavaScript interact with HTML.

    Selecting Elements

    const button = document.getElementById("myButton"); const items = document.querySelectorAll(".item");

    Changing Content & Styles

    button.textContent = "Click Me!"; button.style.backgroundColor = "blue"; button.classList.add("active");

    Event Listeners

    button.addEventListener('click', function() { alert("Button clicked!"); }); input.addEventListener('keypress', (e) => { if (e.key === 'Enter') { submitForm(); } });
